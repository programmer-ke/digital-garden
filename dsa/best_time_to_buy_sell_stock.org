* Best time to buy and sell stock

** Description

https://leetcode.com/problems/best-time-to-buy-and-sell-stock/description/

** Approach

Brute force would be to check that for each possible buy date, the
profit from all possible sell dates. This would be an O(n^{2})
operation.

The sell date always comes after the buy date, and it makes sense to
sell only when price is higher than buy date.

A possible improved approach:

Start with two pointer into the array, the buy followed by the sell pointer
pointing to the first two elements.

As long as it is possible to make a profit, record the profit and
advance the sell pointer.

When we encounter a sell position where we cannot get a profit, i.e.
the buy points to a price greater than or equal to the selling price,
advance the buy pointer to the sell pointer position and the sell
pointer by one. At this point, the buy position is the best possible
one encountered so far.

Repeat the process until the sell pointer gets to the end of array
and return the max profit encountered.

Why this works: the buy pointer is always seeking the best
buying price until the end of the array.

#+begin_src python
from typing import List

class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        max_profit = 0
        min_price = prices[0]
        sell = 1
        while sell < len(prices):
            profit = prices[sell] - min_price
            if profit > 0:
                if profit > max_profit:
                    max_profit = profit
            else:
                min_price = prices[sell]
            sell += 1
        return max_profit

prices = [7]
print(Solution().maxProfit(prices))
# Output: 0

prices = [7,1,5,3,6,4]
print(Solution().maxProfit(prices))
# Output: 5
