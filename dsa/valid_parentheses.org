* Valid Parentheses

** Description

https://neetcode.io/problems/validate-parentheses/question

** Approach

As we traverse the input string every time we encounter an opening
bracket, we push it onto a stack. When we encounter a closing
bracket, we pop from the stack and if the popped bracket does not
match the closing bracket, it the string is invalid.

If we get to the end of the string and have an empty stack,
then the string is valid.

** Solution

#+begin_src python
class Solution:
    def isValid(self, s: str) -> bool:
        if len(s) % 2:
            return False
        stack = []
        match = {")": "(", "}": "{", "]": "["}
        opening = match.values()
        for c in s:
            if c in opening:
                stack.append(c)
            else:
                if not stack or stack.pop() != match[c]:
                    return False
        return not len(stack)
        

s = "([])"
print(Solution().isValid(s))
# Output: true


s = "([)]"
print(Solution().isValid(s))

s = "()[]{}"
print(Solution().isValid(s))
# Output: false
