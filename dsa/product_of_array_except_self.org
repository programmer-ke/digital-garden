** Approach

The naive solution would be to multiply all the numbers in the input
array, then for each position in the output array, divide the overall
product with the number at that position.

However, division is not allowed.

We notice that if as we traverse forward through the array, at index i,
0 < i < len(arr),
we'll have seen all the numbers from index 0 to i, so we can calculate
the product of arr[0] to arr[i-1].

For each arr[i], we therefore only need the product of
arr[i+1] to arr[len(arr) - 1] to fulfill the requirements.

We notice a pattern where if we traversed the array in reverse, we
would be able to access the missing product.

One possible approach is to traverse the array, and for each
each position arr[i], we store the product of arr[0] to arr[i-1] in
results array. At the end of the forward traversal we have products
for all except the first element. And for all except the last element
we have only a partial product.

We can then traverse the source array in reverse and for each arr[j],
0 <= j < len(arr) - 1
we multiply the value in the results array with the product of
array arr[j+1] to arr[len(arr) - 1]

By doing this all each position in the results array has the product
of all elements except itself in the source array.

** Solution

#+begin_src python

def product_except_self(nums):
    results = [1] * len(nums)

    prod = 1
    for i in range(1, len(nums)):
        prod *= nums[i - 1]
        results[i] *= prod

    prod = 1
    for i in range(len(nums) - 2, -1, -1):
        prod *= nums[i + 1]
        results[i] *= prod

    return results

nums = [1,2,3,4]
# Output: [24,12,8,6]
print(product_except_self(nums))

nums = [-1,1,0,-3,3]
print(product_except_self(nums))
# Output: [0,0,9,0,0]

#+end_src
