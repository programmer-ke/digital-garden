* Valid Palindrome

** Description

https://neetcode.io/problems/is-palindrome

** Approach

Start with pointers in both ends of the string. For each valid check
(alphanumeric characters), if the lowercased characters are equal,
advance the pointers closer together. If this is the case for all
positions while the pointers do not overlap, then it is a valid
palindrome.

** Solution

#+begin_src python

s = "Was it a car or a cat I saw?"

def is_palindrome(s):
    start, end = 0, len(s) - 1
    while not end < start:
        while not end < start and not s[end].isalnum(): end -= 1
        while not start > end and not s[start].isalnum(): start += 1
        if end < start:
            return True
        if s[start].lower() == s[end].lower():
            start += 1
            end -= 1
        else:
            return False
    return True
        
print(is_palindrome(s)) # True

s = "tab a cat"

print(is_palindrome(s)) # False

print(is_palindrome("!!??"))  # True


#+end_src
